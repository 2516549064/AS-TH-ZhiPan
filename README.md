# AS-TH-ZhiPan
# AS智盘 V1.0 正式版
个人物品离线盘点台账管理工具

## 📱 应用功能
- 扫码识别盘点：支持二维码、一维码快速录入物品信息
- 拍照物品建档：拍摄物品图片，完善物品档案信息
- 本地数据管理：物品信息整理、分类归档、台账留存
- 标签打印制作：生成物品专属二维码标签，支持图片/PDF导出
- 数据可视化统计：图表展示物品分类、盘点状态
- Excel表格导出：本地导出物品台账数据文件
- 物品图片归档：统一管理所有物品相关实拍图片

## ✅ 应用特点
- 全数据**本地离线存储**，不上传云端，不泄露隐私
- 适配64位安卓全机型，低占用、运行流畅
- 无恶意捆绑、无后台窃取权限
- 合规搭载第三方广告服务

## 📥 安装教程
1.  点击下方附件，下载 `.apk` 安装包
2.  安卓手机前往设置，开启【允许安装未知来源应用】权限
3.  打开下载好的APK文件，一键安装即可使用
